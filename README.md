# 🚀 Python Development Internship – Level 1  
## Cognifyz Technologies

---

## 📌 Project Overview
This repository contains the completed **Level 1 tasks** for the Python Development Internship conducted by **Cognifyz Technologies**.

The project focuses on building a strong foundation in Python programming through hands-on implementation of basic algorithms, logic-building exercises, and mini applications.

These tasks helped improve problem-solving ability, coding logic, and understanding of core Python concepts.

---

## 🧩 Tasks Implemented

### 🔹 Task 1: String Reversal
A program that accepts a string input from the user and returns the reversed version using Python slicing techniques.

---

### 🔹 Task 2: Temperature Conversion
A program that converts temperature values between Celsius and Fahrenheit based on user input and standard mathematical formulas.

---

### 🔹 Task 3: Email Validation
A function that validates email addresses by checking essential components such as the presence of "@" symbol and a valid domain format.

---

### 🔹 Task 4: Calculator Program
A command-line calculator that performs basic arithmetic operations:
- Addition  
- Subtraction  
- Multiplication  
- Division  
- Modulus  

---

### 🔹 Task 5: Palindrome Checker
A program that checks whether a given string is a palindrome (reads the same forward and backward).
