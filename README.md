# Python Development Internship – Level 1  
## Cognifyz Technologies

---

## 📌 Overview
This repository contains the completed Level 1 tasks for the Python Development Internship offered by **Cognifyz Technologies**.  
The tasks focus on strengthening core Python programming concepts, logical thinking, and problem-solving skills.

---

## 🧩 Tasks Completed

### 🔹 Task 1: String Reversal
A Python program that takes a string input from the user and returns the reversed string using slicing.

### 🔹 Task 2: Temperature Conversion
A program that converts temperature values between Celsius and Fahrenheit based on user input.

### 🔹 Task 3: Email Validation
A function that validates whether a given email address is in a proper format by checking for "@" symbol and domain structure.

### 🔹 Task 4: Calculator Program
A basic calculator that performs arithmetic operations such as addition, subtraction, multiplication, division, and modulus.

### 🔹 Task 5: Palindrome Checker
A program that checks whether a given string reads the same forward and backward.

### 🔹 Task 6: Number Guessing Game
An interactive game where the user attempts to guess a randomly generated number with hints like "too high" or "too low".
